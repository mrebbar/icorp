# ICORP Project

Bu Node.js kodi express server yaratish va test API bilan interaktiv “2 qismli” kod olish jarayonini amalga oshirish uchun yozilgan.

# Eslatma
Bu amaliyotni qilish uchun manga ngrok yordam berdi. Ngrok — bu local serveringizni internetga vaqtincha ochish imkonini beruvchi vosita. Agarda siz ham ushbu koddan foydalanmoqchi bo'lsangiz test qilish uchun ngrokdan foydalanishingiz mumkin. Ngrokdan ro'yxatdan o'tasiz va Ngrok sizga token berda. Tokenni pc ga saqlab olasiz. So'ng ko'rsatilgan port bo'yicha server yoqishiz mumkin. Server yoqanizdan keyin u sizga Forwarding URL beradi shu url ni meni codimdagi "myUrl" degan joyiga qo'yib va ortidan /callback deb yozishingiz kerak



# Vazifa bajarish tartibi:

1. https://test.icorp.uz/interview.php  urliga msg va url maydonlaridan iborat JSON formatdagi habar yuboridim yani (POST) qildim. Va shu zahoti kodni 1 qismini yani (part1) ni oldim va  kodni 2 qismi man yuborgan url ga yuborildi 

2. part1 va part2 qismlarini qo'shgan holda yana https://test.icorp.uz/interview.php saytga (GET) so'rov qildim Natijada o'zim yuborgan msg habarimni oldim.
